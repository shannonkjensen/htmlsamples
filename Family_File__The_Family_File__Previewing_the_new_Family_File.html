Previewing the new Family File

At Henry Schein One, we're redesigning the Family File to make it more intuitive and easier to use. You can preview the proposed changes.

To preview the new Family File

1.   Open the Family File and select a patient.

2.   To try the new Family File, click the Try the New Family File toggle.

The new Family File window appears.

Notes:

·        The new Family File window consists of several cards (Contact Information, Patient Information, Continuing Care, Primary Dental Insurance, and so on).

·        You can navigate the Family File controls by pressing the Tab key.

3.   To edit the contents of a card, click the Edit icon.

Note: If a card contains information, the Edit icon appears in the upper-right corner of the card.

4.   Click the Edit icon.

The dialog box corresponding to the card appears.

Note: If a card does not contain any information on file, you can add the information.

5.   For example, to add a referral, click Add Referral.

Dentrix Patient Referrals dialog box appears.

6.   To provide your thoughts of and suggestions for the new Family File design, please click the feedback link.
