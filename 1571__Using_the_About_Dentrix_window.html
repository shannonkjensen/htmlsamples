Using the About Dentrix window

You can access the About Dentrix window from every Dentrix module. This window provides you with basic information about your Dentrix software.

The version number of your Dentrix software (which is a necessary item to have when calling Dentrix Customer Support) is shown in the Version column of the About Dentrix window. The first two digits of the version number indicate the year the version was released. The third digit indicates the individual release for the specified year; for example, the version number 23.2 means that version is the second release of 2023. The remaining digits in the version number indicate the specific build of the released version.

To display the About Dentrix window

·        From the Help menu, click About Dentrix.

What's available in the About Dentrix window?

The About Dentrix window displays other information about your specific installation of Dentrix. The following are descriptions of the information and features available in the About Dentrix window:

·        Dentrix Applications - The installed applications/products list displays all of the Dentrix applications/products installed on your computer.

·        Customer ID - The customer ID is the identification number unique to your practice.

·        Serial Number - The serial number is the number you used to install your software. This is unique for every practice.

·        Installed Version - The version number is the version of each product installed on your computer. The first numerals before the decimal indicate the version.  Any numerals after the first decimal indicate the build of the version.

·        What's New - By clicking the What's New button, you can view brief descriptions of the new features of your version of Dentrix.

·        Check for Updates - By clicking the Check for Updates button, you can access the Dentrix website for the most recent updates of the newest release of Dentrix.

·        System Information - The System Information button opens the System Information window.  This window provides you with information about your computer.

·        Copyrights - The copyright displays the year of the first Dentrix release until the year of your installation. This certifies the rights of the Dentrix software.

·        Terms and Conditions - Click the Terms and Conditions button to review the End User License Agreement.

·        Close - The Close button closes the About Dentrix window.
