Dentrix documentation

The following guides and resources are available to help you install, learn, and use Dentrix in your office:

路        Dentrix System Requirements - The system requirements document describes minimum hardware and software requirements for using Dentrix. Recommendations and notes about using Dentrix with some third-party software and hardware are also included. This document is updated periodically to reflect changes in the market with supported software and hardware.

路        Dentrix Release Guide - The Dentrix software includes a detailed release guide that explains what is included in the software release. The guide is for current Dentrix users who are upgrading to a new version. It includes an overview of new features and enhancements.

路        Dentrix Installation Guide - The Dentrix Installation Guide includes the information you need to know when installing Dentrix. It includes information on how to prepare for installation, step-by-step instructions for installing the software, and information on registering and activating your Dentrix software.

路        Dentrix Help - The most comprehensive Dentrix documentation is provided in the Dentrix Help, which you can access from the Help menu in most Dentrix modules or by pressing the F1 key. The Dentrix Help includes detailed information on all Dentrix modules and features. It provides answers to frequently asked questions, introduces features that were added in recent Dentrix software releases, and provides helpful steps to follow for routine office procedures and much more. The Help is fully indexed and includes advanced full-text search capabilities using the Search icon.

This documentation is provided electronically in several locations depending on the document. Some of these documents are also available in the Dentrix Resource Center. You can access the Dentrix Help from the Help menu in most Dentrix modules. You can print any portion of the documentation using your office printer.
