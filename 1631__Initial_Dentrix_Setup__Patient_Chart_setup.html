Patient Chart setup

You can customize several features of the Dentrix Patient Chart to meet the needs of your practice. You can set up and change these options at any time without affecting data already entered. However, for consistency, it is a good idea to set up at least the following three features on each computer before you begin using Dentrix:

·        Chart colors

·        Procedure buttons

·        A printer
