Ledger setup

There are several features of the Dentrix Ledger that can be customized to meet the needs of your practice. These options can be set up and changed at any time without affecting data already entered. However, for consistency, it is a good idea to set up at least the following three features before you begin using Dentrix:

·        Ledger colors

·        Fast checkout options

·        Direct processing options
