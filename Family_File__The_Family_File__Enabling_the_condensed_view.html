Enabling the condensed view

You can view the redesigned Family File in a “condensed view.” The condensed view reduces the amount of blank space between the individual cards in the redesigned Family File.

To enable the condensed view

1.   Open the new Family File.

The new Family File window appears.

2.   Click the Condensed View toggle.

The condensed view of the redesigned Family File appears.

3.   To revert to the default view, click the Condensed View toggle.
